Список алгоритмов и структур данных, а также их применение в контестах по программированию.
# Графы
### DFS
- обход графа
- поиск компонент связности
- поиск мостов
- поиск точек сочленения
- проверка на предка
- топологическая сортировка
- поиск компонент сильной связности
- построение конденсации графа
### BFS
- обход графа
- поиск компонент связности
- поиск кратчайшего пути в невзвешенном графе
- поиск кратчайшего пути в графе с натуральными весами от 1 до k
### Дейкстра
- поиск кратчайшего пути во взвешенном графе без отрицательных ребер
### Форд-Беллман
- поиск кратчайшего пути во взвешенном графе с отрицательными ребрами
- проверка на отрицательный цикл
### Флойд-Уоршелл
- поиск кратчайшего пути между всеми парами вершин
- поиск отрицательного цикла
### Прим
- минимальное остовное дерево
### Краскал
- минимальное остовное дерево
# Структуры данных
### Система непересекающихся множеств
- поддержка компонент связности графа
### Дерево отрезков
- поиск максимального/минимального элемента на подотрезке
- поиск суммы на подотрезке
- прибавление и присвоение на отрезке
### Декартово дерево
- выполнение операций над множеством за логарифм
### Декартово дерево по неявному ключу
- выполнение операций над массивом за логарифм
### Дерево Фенвика
- вычисление суммы на n-мерном подотрезке с возможностью модификации за логарифм
### Sparse Table
- минимум/максимум/gcd на подотрезке
# Геометрия
### Векторы
- поиск расстояния между структурами
- нахождение пересечения
- поворот
- нахождение угла
- проверка точки на принадлежность многоугольнику
- площадь многоугольника
### Уравнение прямой
- пересечение окружностей
### Грэхем
- минимальная выпуклая оболочка
### Многоугольники
- проверка точки на принадлежность любому многоугольнику
- проверка точки на принадлежность выпуклому многоугольнику
# Динамика
### Классическая
- число путей на n-мерном прямоугольнике

- наибольшая общая подпоследовательность
- игры
### По подмножествам
- разбиение на пары наибольшего числа вершин
- поиск гамильтонова пути
- замощение прямоугольника доминошками
